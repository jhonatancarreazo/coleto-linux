# Arquitectura de Coleto CLI

## Lenguaje
Python 3.12+

# Framework CLI

Typer

## Filosofía

Coleto CLI será una capa amigable sobre las herramientas nativas de Debian.
Nunca reemplazará apt, dpkg ni otras utilidades del sistema.
Su objetivo es ofrecerun experiencia intuittiva y cercana para el usuario.

## Compatibilidad

- Debian Stable
- Derivados de Debian

## Objetivos

- Codigo limpio
- Modular
- Facil de extender
- Facil de probar
